# Simple Todo Application

A full-stack Todo application built with **Spring Boot** and **React**.

## Tech Stack
- **Backend:** Java 17+, Spring Boot, JPA, H2 (In-memory DB)
- **Frontend:** React, Axios, CSS3

## How to Run
1. **Backend:**
   - Navigate to `/backend`
   - Run `./mvnw spring-boot:run`
   - API is available at `http://localhost:8080/api/todos`

2. **Frontend:**
   - Navigate to `/frontend`
   - Run `npm install`
   - Run `npm start`
   - UI is available at `http://localhost:3000`

## Features
- CRUD operations (Create, Read, Update, Delete)
- Cross-Origin Resource Sharing (CORS) enabled
- In-memory data storage (H2)
